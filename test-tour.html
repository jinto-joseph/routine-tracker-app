📝 How to Test the Tour

  1. Click "Reset Tour" to clear the completion flag
  2. Click "Open Main App" to go to the main application
  3. The tour should start automatically after 2 seconds
  4. If it doesn't start, check the browser console (F12) for errors

🔍 Console Commands

Open browser console (F12) and try these:

  • window.restartHabitFlowTour() - Restart tour and reload
  • window.habitFlowTour - Access tour object
  • localStorage.getItem('habitflow_tour_completed') - Check completion status
